# 📊 Financial Analytics Dashboard

## 📌 Project Overview

This project is an interactive Financial Analytics Dashboard built in Microsoft Power BI to analyze business performance through key financial metrics and interactive visualizations.

---

## 🎯 Project Objective

To provide business users with a comprehensive view of financial performance by analyzing sales, profit, cost, and customer insights, enabling better business decisions.

---

## 🛠 Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX
- Microsoft Excel

---

## 📈 Dashboard Features

- KPI Cards
- Sales Analysis
- Profit Analysis
- Cost Analysis
- Interactive Slicers
- Drill-through Analysis
- Dynamic Charts
- Business Insights

---

## 📊 KPIs

- Total Sales
- Total Profit
- Total Cost
- Total Orders
- Average Sales
- Average Profit

---

## 📚 Skills Demonstrated

- Data Cleaning
- Data Transformation
- Data Modeling

- Dashboard Design
- Data Visualization
- Business Intelligence
